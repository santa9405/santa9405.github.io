# 1013 : [기초-입출력] 정수 2개 입력받아 그대로 출력하기
> [입력] 1 2<br/>
  [출력] 1 2

```java
import java.util.Scanner;
public class Main {
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int input1 = sc.nextInt();
		int input2 = sc.nextInt();
		System.out.print(input1 + " " + input2);
	}	
}
```
